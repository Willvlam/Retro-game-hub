# 🎮 Retro Game Hub

A collection of classic arcade games playable directly in your browser!

## Games Included

### 🪨 Asteroids
Destroy asteroids while avoiding collisions. Classic arcade action!
- **Controls**: Arrow Keys to move, Space to shoot
- **Goal**: Destroy all asteroids to advance levels
- **Game Over**: Lose all 3 lives and it's game over

### 🐍 Snake
The timeless snake game - eat food and grow!
- **Controls**: Arrow Keys to move
- **Goal**: Eat the red food to grow and score points
- **Game Over**: Hit a wall or yourself

### 🎾 Pong
Two-player paddle action classic
- **Controls**: 
  - Player 1: W/S keys
  - Player 2: Up/Down arrow keys
  - Space to start
- **Goal**: Be the first to score enough points

## How to Play

1. Open `index.html` in a web browser
2. Select a game from the menu
3. Follow the on-screen instructions
4. Press R to restart when the game ends (Asteroids and Snake)
